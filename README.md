# Quantum Simulations with QuTiP

## Overview

This repository demonstrates a collection of quantum mechanics and quantum computing simulations using the QuTiP framework.

## Topics Covered

- Bloch Sphere Visualization
- Quantum Gates and Rotations
- Rabi Oscillations
- Open Quantum Systems
- Decoherence and Relaxation
- Jaynes-Cummings Model
- Quantum Circuits
- Schrödinger Cat States
- Squeezed Vacuum States
- Wigner Functions
- Landau-Zener Dynamics
- Resonance Spectroscopy

## Tools

- Python
- QuTiP
- NumPy
- Matplotlib

## Main Notebook

- Quantum_Simulations.ipynb
